# Plataforma Georreferenciada de Incidencias Caninas

## Descripción

Prototipo funcional de una plataforma de participación ciudadana para el monitoreo territorial de incidencias relacionadas con la población canina en situación de calle en la provincia Cercado del municipio de Tarija.

## Tecnologías

### Frontend

- HTML5, CSS3, Bootstrap 5
- JavaScript Vanilla
- Leaflet (mapas)
- Chart.js (estadísticas)

### Backend

- Node.js + Express.js
- PostgreSQL (remoto)

## Estructura del Proyecto

proyectofinal_plataforma/
├── backend/ # API REST

├── base_datos/ # Scripts SQL
